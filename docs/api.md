# API

Static API artifacts are generated by `scripts/generate-scores.ts` and published under `public/api`.

Base path on GitHub Pages:

- `/trust-dashboard/api/*`
- `/trust-dashboard/openapi.json`

## Endpoints

- `/api/index.json`
- `/api/leaderboards/milady/lifetime.json`
- `/api/leaderboards/eliza/lifetime.json`
- `/api/leaderboards/eliza/weekly.json`
- `/api/leaderboards/eliza/monthly.json`
- `/api/leaderboards/eliza-effect/lifetime.json`
- `/api/contributors/{username}/profile.json`
- `/api/repos/index.json`

## Response Envelope

Leaderboard responses use a consistent envelope:

```json
{
  "version": "1.0",
  "source": "milady-ai/milaidy",
  "period": "lifetime",
  "generatedAt": "2026-02-22T00:00:00.000Z",
  "totalUsers": 123,
  "leaderboard": []
}
```

## Discovery and Aliases

`/api/index.json` includes direct endpoint links and aliases:

- `aliases.defaultBoard`
- `aliases.milady`
- `aliases.eliza`
- `aliases.elizaEffect`

## OpenAPI

Machine-readable contract:

- `/openapi.json`

This spec is intentionally lightweight and stable for static hosting.
